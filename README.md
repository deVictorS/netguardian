# Projeto: NetGuardian - Inteligência de Ameaças com Python e SQL #

</div>

---

## Sobre o projeto

Este projeto é um **sistema de Threat Intelligence**, desenvolvido em **Python**, integrado com **MySQL** e bibliotecas de **análise de dados**. O software automatiza a coleta de endereços IPs do site bloacklist.de, processa esses dados, armazena em um banco de dados relacional  gera um gráfico de pizza para análise visual. Além disso, possui uma interface de consulta de ameaças e IPs.
